DR Fair Price Monitor — เทียบราคาจริง vs Fair Price (Arbitrage)

Fair Price = ราคาหุ้นแม่ × อัตราแลกเปลี่ยน ÷ อัตราแปลงสภาพ (ratio) — ดึงราคาจริง+ratio สดจาก SET, ราคาหุ้นแม่/FX จาก Yahoo Finance

อัปเดต:

⚠️
ข้อมูลเป็น snapshot ไม่ใช่ real-time
ราคาและ ratio ดึงจาก SET (ราคาปิดล่าสุด) + ราคาหุ้นแม่/FX จาก Yahoo Finance ณ เวลาที่ scan — ต้องเทียบกับราคาจริงในกระดานอีกครั้งก่อนตัดสินใจซื้อขายจริงเสมอ ครอบคลุม 222 DR ที่รวบรวมได้จากไฟล์ในโปรเจกต์ + watchlist ผู้ใช้ + ที่พบเพิ่มเติม (ยังไม่ใช่ DR ทั้งหมดที่มีใน SET ซึ่งมีมากกว่านี้ — SET ไม่มี bulk list ให้ดึง ต้องพบทีละตัว — ยังมี ~36 ตัวที่หา mapping หุ้นแม่ไม่ได้ ส่วนใหญ่เป็น ETF/index DR และหุ้นตลาดรอง)

🛑 STOP — จุดตรวจสอบก่อนเชื่อ
  • Ratio ทุกตัวดึงสดจาก SET (`exerciseRatio` ในหน้า factsheet ของแต่ละ DR) ไม่เคยใช้ค่าที่เก็บไว้เก่าเลย — ระหว่างพัฒนาเวอร์ชันแรก (39 DR) พบว่าไฟล์เก่าของโปรเจกต์ (raw/dr/) มี ratio ผิด/ล้าสมัยถึง 7 ตัว (เช่น CRWD80: เก่า 4,000 จริง 250 — ถ้าใช้ค่าเก่าจะได้ Premium ผิดเป็น +1500% แทนที่จะเป็น +0.15% จริง) จึงยึดหลักไม่เชื่อไฟล์เก็บไว้แบบ static เด็ดขาด
  • ราคาหุ้นแม่และอัตราแลกเปลี่ยนมาจาก Yahoo Finance chart API (public, ไม่ใช้ API key) — อาจมี delay 15-20 นาทีสำหรับบางตลาด
  • ราคา DR จาก SET เป็นราคาปิดล่าสุด (`marketDateTime` ต่อแถว) ไม่ใช่ real-time ระหว่างเวลาทำการ
  • ครอบคลุม 222 DR ที่รวม mapping หุ้นแม่ได้ (จากไฟล์ raw/dr/ + 2 TradingView watchlist ของผู้ใช้ "doang_DR_China"/"doang_DR_US" + SET_DR_tradingview_import + ตัวที่ผู้ใช้พบเพิ่มเติมผ่านการค้นหาบน set.or.th เช่น MSFT03/19/23, MICRON01/03/19/80) — ยังมีอีก ~36 DR ที่หา Yahoo Finance ticker ของหุ้นแม่ไม่ได้ (ส่วนใหญ่เป็น ETF/index DR เช่น กองทุนทอง/น้ำมัน/ดัชนีประเทศ และหุ้นตลาดรอง เช่น เวียดนาม ไต้หวัน) — ไม่รวมเพราะไม่อยากเดา ticker ผิด ไม่ใช่เพราะไม่มีอยู่จริง — ไม่มี bulk list ของ DR ทั้งหมดจาก SET ให้ดึง (ลอง sitemap.xml แล้วไม่มี, ลอง API endpoint แล้วโดน WAF บล็อก) coverage จะเพิ่มทีละตัวเมื่อพบเจอเพิ่ม
  • 3 ตัว (ADVANT23, NESTLE80, SYNP23) มี Yahoo ticker แต่ดึงราคาไม่สำเร็จระหว่าง scan — แสดงราคาจริงได้แต่ไม่มี Fair Price
  • หน้านี้เป็น static snapshot (ไม่ auto-refresh) — ต้องขอให้ scan ใหม่ (`.tools/scan_dr.py`) เพื่ออัปเดตราคา
  • Premium/Discount ที่เห็นเป็นตัวเลขดิบ ยังไม่หักค่าธรรมเนียมซื้อขาย/สเปรด/ภาษี — ต้องคำนวณต้นทุนจริงเพิ่มก่อนตัดสินใจ arbitrage

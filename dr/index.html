DR Fair Price Monitor — เทียบราคาจริง vs Fair Price (Arbitrage)

เทียบราคา DR ที่เทรดจริงบน SET กับ Fair Price ที่คำนวณจากราคาหุ้นแม่ × อัตราแลกเปลี่ยน ÷ อัตราแปลงสภาพ (ratio) — ใช้ TradingView live chart ทั้งสองฝั่ง

ที่มา: raw/dr/SET_DR_with_Ratios_and_FairTHB_2025-09-11.csv, raw/dr/DR_fairprice_template.xlsx, TradingView watchlists "doang_DR_China"/"doang_DR_US" (เฉพาะรายชื่อ ticker ไม่ใช่ ratio) — อัปเดต 2026-07-12

⚠️
Ratio ต้องยืนยันเองเสมอ ห้ามเดา
มีเพียง 39 DR ที่มี ratio ยืนยันแล้ว (จาก raw/dr/) — DR อีก ~150 ตัวที่เห็นชื่อจาก watchlist ของคุณ ยังไม่มี ratio ในระบบ ต้องกรอกเองในฟอร์มของแต่ละแถว (จาก SET DR factsheet หรือแหล่งที่คุณเชื่อถือ) ก่อนจะเห็น Fair Price — ค่าที่กรอกจะถูกจำไว้ในเบราว์เซอร์นี้ (localStorage) เท่านั้น ไม่ได้ส่งกลับมาที่ repo อัตโนมัติ
⚠️ BABA80 มีข้อมูล ratio ขัดแย้งกันระหว่าง 2 ไฟล์ในโปรเจกต์ (100 vs 1000) — แสดงเป็น ⚠️ conflict ไม่ใช้ตัวเลขใดจนกว่าจะตรวจสอบ

  • Ratio ของ 39 DR ที่มีเครื่องหมาย ✓ มาจาก raw/dr/SET_DR_with_Ratios_and_FairTHB_2025-09-11.csv และ raw/dr/DR_fairprice_template.xlsx — CATL01 (ratio 440) เป็นค่าที่คุณให้มาเอง ยังไม่ได้เทียบกับ SET factsheet โดยตรง (แต่ cross-check คร่าวๆ จากราคาจริงใน watchlist ของคุณ ณ ขณะ ingest ให้ผลใกล้เคียง 440 ที่ FX ปัจจุบัน)
  • BABA80 มีข้อมูลขัดแย้ง (CSV=100 vs xlsx=1000) — ห้ามเชื่อค่าใดค่าหนึ่งจนกว่าจะตรวจสอบกับ SET DR factsheet จริง
  • DR อีก ~150 ตัว ชื่อ ticker มาจาก TradingView watchlist ของคุณเอง (doang_DR_China, doang_DR_US) แต่ ratio/ตลาดหุ้นแม่ยังไม่มีในระบบ — ต้องกรอกเองทีละตัว ค่าที่กรอกเก็บใน localStorage ของเบราว์เซอร์นี้เท่านั้น (ถ้าเปลี่ยนเครื่อง/ล้าง cache จะหายและต้องกรอกใหม่)
  • Fair Price chart ใช้ TradingView symbol formula (เช่น NASDAQ:AAPL*FX_IDC:USDTHB/1000) ผ่าน embed widget — เคยพบว่า TradingView embed widget "ปฏิเสธ" สูตรที่ซับซ้อนเกินไป (เช่น การหารด้วยราคาจริงในสูตรเดียว) จึงต้องแยกเป็น 2 กราฟ (ราคาจริง, Fair Price) แทนที่จะรวมเป็นกราฟอัตราส่วนเดียว — ถ้ากราฟฝั่งใดฝั่งหนึ่งไม่ขึ้น ให้กดลิงก์ "เปิดอัตราส่วนที่ TradingView" เพื่อดูแบบเต็มที่เว็บ TradingView โดยตรง
  • ไม่มีตัวเลข Premium/Discount % คำนวณให้อัตโนมัติในหน้านี้ (ไม่สามารถอ่านค่าตัวเลขจาก TradingView iframe ข้าม origin ได้) — ต้องเทียบด้วยตาจาก 2 กราฟ หรือกดลิงก์ไปดูอัตราส่วนที่ TradingView โดยตรง

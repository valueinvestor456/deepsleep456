🛑 STOP — จุดตรวจสอบก่อนเชื่อ
  • Ratio ทุกตัวดึงสดจาก SET (`exerciseRatio` ในหน้า factsheet ของแต่ละ DR) ไม่ใช้ค่าที่เก็บไว้เก่าอีกต่อไป — ระหว่างพัฒนาพบว่าไฟล์เก่าของโปรเจกต์ (raw/dr/) มี ratio ผิด/ล้าสมัยถึง 7 จาก 39 ตัว (ADBE06, BKNG80, CRWD80, MS06, NFLX80, VISA80, SIA19) ซึ่งแก้ไขให้ถูกต้องแล้วในหน้านี้โดยอิงจาก SET โดยตรง
  • ราคาหุ้นแม่และอัตราแลกเปลี่ยนมาจาก Yahoo Finance chart API (public, ไม่ใช้ API key) — อาจมี delay 15-20 นาทีสำหรับบางตลาด
  • ราคา DR จาก SET เป็นราคาปิดล่าสุด (`marketDateTime` ต่อแถว) ไม่ใช่ real-time ระหว่างเวลาทำการ
  • ครอบคลุมแค่ 39 DR ที่มีการ scan ไว้ล่วงหน้า — ยังไม่ใช่ DR ทั้งหมดที่มีขายบน SET (มีอีกหลายร้อยตัว) เพราะการดึงข้อมูลแต่ละตัวต้องทำทีละตัว (ไม่มี API list รวม) — ถ้าต้องการเพิ่ม DR ตัวอื่น ต้องแจ้งให้ scan เพิ่ม
  • หน้านี้เป็น static snapshot (ไม่ auto-refresh) — ต้องขอให้ scan ใหม่เพื่ออัปเดตราคา
  • Premium/Discount ที่เห็นเป็นตัวเลขดิบ ยังไม่หักค่าธรรมเนียมซื้อขาย/สเปรด/ภาษี — ต้องคำนวณต้นทุนจริงเพิ่มก่อนตัดสินใจ arbitrage

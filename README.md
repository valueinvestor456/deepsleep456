# Deepsleep456 — เว็บไซต์ที่พัก

เว็บไซต์หน้าเดียว (static HTML/CSS/JS ไม่มี build tools) สำหรับที่พัก Deepsleep456
ตำบลโป่งแยง อำเภอแม่ริม จังหวัดเชียงใหม่

## วิธีเปิดดู

เปิด `index.html` ได้โดยตรง (ดับเบิลคลิก) หรือรันเซิร์ฟเวอร์ในเครื่อง:

```bash
python -m http.server 8080
```

แล้วเข้า `http://127.0.0.1:8080`

## โครงสร้างไฟล์

```
index.html         หน้าเว็บหลักทั้งหมด (8 ส่วนตามบรีฟ)
css/style.css       สไตล์ทั้งหมด
js/main.js          เมนูมือถือ, สไลด์ hero, ตัวกรองแกลเลอรี, lightbox
images/hero/        รูป hero (คืนกลางแจ้ง/มุมสูง)
images/rooms/       ห้องนอน 3 ห้อง
images/bathroom/    ห้องน้ำ + อ่างอาบน้ำ indoor/outdoor
images/common/      พื้นที่ส่วนกลาง (นั่งเล่น/ระเบียง)
images/exterior/    ภายนอกบ้าน/มุมสูง
images/og/          รูปสำหรับแชร์โซเชียล
images/logo.png     โลโก้จริงของ Deepsleep456 (พื้นหลังโปร่งใส)
```

ภาพทั้งหมดคัดและบีบอัดมาจาก `Deepsleep/Deepsleep webpage/` และ `Deepsleep/photoes deepsleep/`
(โฟลเดอร์ต้นฉบับที่มีรูปดิบกว่า 280 ไฟล์) ผ่าน ffmpeg เพื่อลดขนาดไฟล์ให้เว็บโหลดเร็ว

## SEO Keywords ที่ใช้ (จากการวิเคราะห์คู่แข่งรัศมี 5–10 กม. — ก.ค. 2569)

เรียงตามลำดับความสำคัญ ใช้ใน title / meta / เนื้อหาของทั้งสองภาษาแล้ว:

1. ที่พักโป่งแยง — แข่งต่ำ ควรเป็นของเรา (title + H1)
2. ที่พักใกล้ม่อนแจ่ม — ปริมาณค้นสูงสุดของย่าน (title + H1)
3. ที่พักม่อนแจ่ม มีอ่างอาบน้ำ — มีบทความลิสต์รองรับ = ดีมานด์จริง
4. ที่พักม่อนแจ่ม หมูกระทะ — มีบทความลิสต์รองรับ
5. ที่พักม่อนแจ่ม วิวทะเลหมอก — คำขายอันดับ 1 ของย่าน
6. บ้านพักทั้งหลัง เชียงใหม่ กลุ่มใหญ่ 12 คน — intent จองสูงสุด
7. ที่พักแม่ริม วิวดอย
8. ที่พักเชียงใหม่ อากาศเย็น หนีร้อน — section climate รองรับ
9. ที่พักม่อนแจ่ม คาราโอเกะ — คู่แข่งเกือบศูนย์
10. (EN) private house near Mon Cham / Chiang Mai mountain villa 12 guests — หน้า /en/

งานตลาดถัดไปที่แนะนำ: ทำ Google Business Profile, ส่งที่พักเข้าบทความลิสต์ (chillpainai/paknaidi), ทำคลิปสั้น TikTok/Reels
